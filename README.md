# 📚 Chat with Docs

AI-powered document chat application built with
LangChain, Pinecone and Gemini AI.
Upload PDFs or add URLs and chat with your documents!

## 🚀 Live Demo

https://chat-with-docs-rag-git-master-nishantaisharma004-8526s-projects.vercel.app/

## ✨ Features

- 📄 Upload multiple PDFs
- 🔗 Add URLs to chat with
- 🧠 Conversation memory
- ⚡ Streaming responses
- 🔍 Semantic search with Pinecone
- 🤖 Powered by Google Gemini AI

## 🛠️ Tech Stack

- Next.js 14
- LangChain.js
- Pinecone Vector DB
- Google Gemini AI
- Vercel AI SDK v6
- Tailwind CSS

## 📦 Setup

```bash
git clone https://github.com/yourusername/chat-with-docs
cd chat-with-docs
npm install --legacy-peer-deps
```

Create `.env.local`:

```
GOOGLE_GENERATIVE_AI_API_KEY=your_key
PINECONE_API_KEY=your_key
PINECONE_INDEX=chat-with-docs
```

```bash
npm run dev
```

## 🧠 Concepts Used

- RAG (Retrieval Augmented Generation)
- Vector Embeddings
- Semantic Similarity Search
- Conversation Memory
- Question Rephrasing
- LCEL (LangChain Expression Language)
- Streaming Responses
